## 🚀 Features

- Multiple choice questions
- Score tracking
- Progress bar
- Result screen with performance message
- Restart functionality
- Responsive design

## 📂 Project Structure

Quiz-Game/
│
├── index.html
├── style.css
├── script.js
└── README.md

## 💡 How It Works

- Questions are stored in a JavaScript array.
- DOM manipulation is used to dynamically render questions.
- Score updates based on correct answers.
- Results are displayed at the end.

## 🧠 What I Learned

- DOM manipulation
- Event handling
- Dynamic UI updates
- Progress tracking logic
- Conditional rendering
